寒假作业
=======


实现的功能
------
* 用户登陆注册
* 用户上传视频
* 用户添加评论
* 拉取用户所传的视频
* 视频分类
* 拉取视频下的评论
* 搜索视频


接口概述
-----

var settings = {
  "async": true,

    "Cache-Control": "no-cache",
